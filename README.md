# gbf-manager

グラブルの育成・素材管理を自分用にまとめたリポジトリです。

---

## 📄 収録ツール

### 1. 総合素材・進捗管理 (`index.html`)
* **素材管理:** クォーツ、属性晶核、天星の欠片の不足分計算。
* **十天衆 限界超越:** Lv110〜150までの必要素材リスト化と進捗管理。

### 2. 神石超越・最適化計算機 (`optimas.html`)
* **最適化シミュレーション:** 神石（オプティマスシリーズ）超越において、手持ちの素体数から「金剛晶の消費を抑えるか」「アニマ確保を優先するか」の最適ルートを算出します。

---

## 🤖 AI-Assisted Development
このプロジェクトのロジック構築およびコードの実装には **AI (Google Gemini)** を活用しています。
AIとの対話を通じて、複雑な素材計算アルゴリズムの生成やUIの最適化を行いました。

---

## 🛠 使い方・仕様
* **起動:** 各HTMLファイルをブラウザで直接開くことで動作します。
* **保存:** 入力データはブラウザの `LocalStorage` に保存されます。
* **構成:** 外部ライブラリに依存しない Vanilla JS で構成されているため、オフラインでも動作可能です。
